Configuration

General

This addon is preconfigured and should work out of the box upon installation.

Archive manager

The most important configuration is about selecting the Web Archive manager to be used. This is governed by dependency management in your pom, please see install page.

Manager Parameters

Depending on the manager used, different parameters will be exposed. The configuration can be found under /hippo:configuration/hippo:modules/web-archiving-addon-services-daemon/archivemanager

HST URL service

This addon exposes a service in HST that fetches all URLs of a document. Configuration for this service can be found in hst-url-service/src/main/resources/META-INF/hst-assembly/addon/org/onehippo/forge/webarchiving/hst/web-archiving-hst.xml. You can change the initialization parameters of the "org.onehippo.forge.webarchiving.hst.service.HstUrlServiceImpl" bean as needed
